## element 对象

jsx 方法用来生成 element 对象，它执行的结果才是 element 对象，
`element` 对象是对 `UI` 的描述

## fiber 对象

fiber 对象是对 react 执行过程中的描述，是一个链表结构，用来描述组件的状态和结构
